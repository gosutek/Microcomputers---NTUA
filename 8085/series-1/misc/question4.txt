καμπυλη διακριτων στοιχειων / IC = (20000 + 20x) / x
καμπυλη FPGA = (10000 + 40x) / x
καμπυλη SoC-1 = (100000 + 4x) / x
καμπυλη SoC-2 = (200000 + 2x) / x

εξισωνουμε για να βρουμε τομες

IC-FPGA: x = 500
IC-SoC-1: x = 5000
IC-SoC-2: x = 10000
FPGA-SoC-1: x = 2500
FPGA-SoC-2: x = 5000
SoC-1-SoC-2: x= 50000

απο το διαγραμμα βγαζεις σε ποιες περιοχες ειναι ελαχιστα
